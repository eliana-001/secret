# Carta de San Valentín

Página estática creada con HTML, CSS y JavaScript puro.

Cómo usar:

- Abrir `index.html` en un navegador moderno.
- Navegar entre las tres vistas con los botones en la parte superior: `Carta`, `Carrusel`, `Cosas amorosas`.
- En la vista `Carta` puedes escribir un texto personalizado y abrir/cerrar el sobre.
- En `Carrusel` las imágenes cambian automáticamente; usa ‹ › o los puntos para controlar.
- En `Cosas amorosas` puedes generar mensajes y copiarlos al portapapeles.

¡Feliz San Valentín!
