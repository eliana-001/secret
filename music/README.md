# Cómo cambiar la música

1. Descarga tu canción romántica favorita en formato MP3
2. Renómbrala como `romantic.mp3`
3. Colócala en la carpeta `music/`
4. ¡Listo! La música se reproducirá automáticamente

## Alternativa: Usar otra canción con diferente nombre

Si quieres usar un archivo con otro nombre (por ejemplo `mi-cancion.mp3`):

1. Coloca tu archivo MP3 en la carpeta `music/`
2. Abre el archivo `index.html`
3. Busca la línea: `<source src="music/romantic.mp3" type="audio/mpeg">`
4. Cámbiala por: `<source src="music/mi-cancion.mp3" type="audio/mpeg">`
